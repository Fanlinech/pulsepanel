# PulsePanel

PulsePanel is a backend API for managing and monitoring servers.

## Tech Stack

- .NET 10
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- Docker

## Current Features

- Server CRUD
- Heartbeat endpoint
- Online / Offline status calculation
- Dashboard summary

## Project Status

Work in progress.
